# Bus Management System (Hibernate Project)

## Steps to Run
1. Open Eclipse IDE.
2. Go to File → Import → Maven → Existing Maven Projects → Select this folder.
3. Run MySQL and create database:
   ```sql
   CREATE DATABASE busdb;
   ```
4. Open `MainApp.java` and Run as → Java Application.
5. Perform CRUD operations from console.

## Features
- Add new bus details
- View all buses
- Search bus by route
- Update bus capacity
- Delete bus
